# Student Productivity Dashboard

A responsive productivity dashboard built using React.js, JavaScript, Tailwind CSS, and Recharts to help students track study sessions, monitor habits, analyze productivity trends, and visualize progress.

## Features

- Study Session Management (Add / Delete)
- Habit Tracking
- Goal Progress Monitoring
- Focus Score Calculation
- Productivity Analytics Dashboard
- Weekly Study Trends
- Subject-wise Study Distribution
- Local Storage Persistence
- Responsive UI Design

## Tech Stack

- React.js
- JavaScript
- Tailwind CSS
- Recharts
- Local Storage

## Key Metrics

- Total Study Hours
- Average Study Hours
- Goal Progress
- Focus Score
- Active Study Days
- Habit Completion Rate
- Subject-wise Performance

## Installation

```bash
npm install
npm start
```
